# 🍵 Tea IDE

A clean, browser-based IDE for the Tea programming language. Write Tea, compile to Python, and run it — all in your browser.

## ✨ Features

- 🎨 Dark theme editor with line numbers
- 🔨 Compile Tea → Python with one click
- ▶️ Run Tea programs in-browser
- 📥 Code templates (Hello World, Input, Variables, Loops, Functions)
- 💾 Save and open `.tea` files
- 📱 Mobile-friendly (works on iPhone/iPad)
- 💾 Auto-saves to your browser

## 🚀 Quick Start

1. Open `index.html` in your browser
2. Write Tea code (or use a template)
3. Click **▶ Run** or **🔨 Compile**

No installation. No server. Just open and code.
